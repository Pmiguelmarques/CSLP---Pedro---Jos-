Detailed Description

A classe CBF implementa um Counting Bloom Filter, ou seja, implementa uma estrutura de dados onde é possível adicionar elementos ao seu array, de tamanho n, dado um número de funções de hash k,. Um elemento é sujeito a k funções de hash, sendo que é calculada a posição que esse elemento irá ocupar no array, para cada função de hash, ou seja, o elemento irá ser inserido em k posições
